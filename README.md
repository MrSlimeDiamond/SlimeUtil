# SlimeUtil
Utilities for Java

## Usage
### Gradle (Groovy)
```groovy
repositories {
    maven {
        url 'https://repo.zenoc.net/repository'
    }
}

dependencies {
    implementation 'net.slimediamond:util:1.0-SNAPSHOT'
}
```

### Gradle (Kotlin)
```kotlin
repositories {
    maven("https://repo.zenoc.net/repository")
}

dependencies {
    implementation("net.slimediamond:util:1.0-SNAPSHOT")
}
```
